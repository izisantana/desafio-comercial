# Dashboard Comercial

Projeto de análise comercial desenvolvido com **Python** para tratamento de dados e **Power BI** para modelagem e visualização.

## Objetivo

Construir um dashboard interativo para acompanhar o desempenho comercial, permitindo analisar:

- faturamento total;
- desempenho das lojas em relação às metas;
- histórico de vendas por vendedor;
- análise salarial dos colaboradores;
- ticket médio por loja;
- indicadores complementares para apoio à decisão.

---

## Tecnologias utilizadas

- **Python**
- **Pandas**
- **Power BI**
- **Excel**
